# 🏏 Cric Opto - Dream11 Cricket Optimization

A comprehensive Streamlit application designed to help Dream11 cricket players achieve first rank by analyzing player statistics, team performance, and creating optimal team combinations.

## 🚀 Features

- **Player Analysis**: Comprehensive cricket player statistics and fantasy point calculations
- **Team Builder**: Interactive team selection with position-based recommendations
- **Optimal Team Generator**: AI-powered team creation using advanced algorithms
- **Strategy Guide**: Expert tips and strategies for Dream11 success
- **Advanced Analytics**: Interactive charts and performance insights
- **Real-time Updates**: Live player data and form analysis

## 🎯 How to Use

1. **Filter Players**: Use the sidebar to filter players by position and minimum fantasy points
2. **Build Team**: Manually select players or use the auto-generate feature
3. **Analyze Performance**: View team composition and total fantasy points
4. **Follow Strategy**: Use the built-in strategy guide for better decisions

## 🏆 Dream11 Cricket Scoring

- **Batting**: Runs, strike rate bonuses, consistency points
- **Bowling**: Wickets, economy rate bonuses, bowling average
- **Fielding**: Catches, stumpings, run-outs
- **Form Bonus**: Recent performance trends and momentum

## 📱 Access

This application is deployed on Hugging Face Spaces and can be accessed directly in your browser.

## 🔧 Technical Details

Built with Streamlit, Pandas, NumPy, and Plotly for a modern, interactive web experience.
